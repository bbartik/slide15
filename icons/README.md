# App Icons

This folder needs icon images for the PWA to work properly on mobile devices.

## Quick Solution: Generate Icons Online

Visit **https://www.pwabuilder.com/imageGenerator** and upload a 512x512 image with your game logo.

It will generate all the icon sizes you need:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Alternative: Use a Simple Colored Square

For testing, you can create simple colored squares using any image editor:

1. Create a 512x512 image
2. Fill it with your brand color (#2a5298 - blue)
3. Add white text "GAMES" in the center
4. Save as icon-512x512.png
5. Resize and save smaller versions

## Icon Requirements

- **Format**: PNG
- **Background**: Solid color (no transparency for splash screens)
- **Design**: Simple and recognizable at small sizes
- **Maskable**: Keep important content in the center 80% of the image

## Screenshots (Optional)

For better app store presentation, add:
- screenshot-wide.png (1280x720) - Desktop view
- screenshot-narrow.png (720x1280) - Mobile view

Just take screenshots of your homepage and games!
